Day 4

Component showcase goes multi-org

Long session across the afternoon and evening, lots going on. Fixed a password bug that was letting people straight through if they pressed Cancel (embarrassing). Moved the whole thing off Netlify after hitting the free tier limit, over to GitHub Pages which is completely free. Then the big one: turned the component showcase into a proper multi-organisation reference tool, going from 34 GOV.UK components to over 110 across 5 government design systems, each with their own styling and a filter to switch between them. Had to install Node.js at one point which felt like a big deal. Later added a navigation bar across the top of every page so you can always get back to where you came from, tidied up some of the copy across the sites, and split the prototypes landing page into separate Prototypes and Documentation sections. Really happy with where it ended up.
